## 🚀 快速开始

如果你现在就想开始，执行以下步骤：

### 1. 注册 Neon
访问：https://neon.tech
点击 "Sign Up" 或 "Get Started"

### 2. 创建项目
- Project name: `mypilot`
- Region: 选择离你最近的
- 点击 "Create Project"

### 3. 复制连接字符串
会显示类似这样的字符串：
```
postgresql://username:password@ep-xxx.neon.tech/neondb?sslmode=require
```

### 4. 更新 .env
打开 `.env` 文件，更新：
```env
DATABASE_URL="你复制的连接字符串"
```

### 5. 运行设置
```powershell
.\scripts\setup-database.ps1
```

### 6. 启动应用
```bash
npm run dev
```

### 7. 访问
http://localhost:3000
